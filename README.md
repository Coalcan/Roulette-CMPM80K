# Roulette-CMPM80K
Roguelike Roulette




If on linux subsystem, don't boot the built in GDScript language server. 

use: 
$ godot --rendering-driver openg13 --edito
to boot up godot engine. It wont lag out your machine


assets/ = imported raw assets (models, textures, audio) — stuff that comes from outside
scenes/ = .tscn files you author (Player, Level, UI, etc.)
scripts/ = .gd files
